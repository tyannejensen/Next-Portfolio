## 🏗 Getting Started

Let's get this party started, shall we?

### Step 1: Clone the repo

```
git clone git@github.com:tyannejensen/portfolio.git
cd portfolio
```

### Step 2: Install dependencies:

```
npm install
```

### Step 3: Run Forrest Run

```
npm run start
```

It is now time to open your localhost and make it your own!

## 🌐 Deployment

Deploying is a breeze with Render. Link your GitHub repo, hit deploy, and watch
your project go live with ease.
